# personal-ai-engineering-lab

This repo is my hands-on AI engineering lab.

Purpose:
Build, test, break, and understand practical AI workflows using GitHub, GitHub Codespaces, OpenAI Codex, GPT models, and agent-style tools.

Focus areas:
- API-first AI systems
- structured outputs
- AI-assisted coding
- evaluation scripts
- hallucination testing
- prompt injection testing
- audit logging
- tool permissions
- human approval flows
- enterprise AI engineering patterns

Operating principle:
Tools are practice surfaces, not identities.

This lab is meant to build real technical fluency through small, serious experiments.
