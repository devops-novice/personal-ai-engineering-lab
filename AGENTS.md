# Repo instructions for AI coding agents

This repo is a personal AI engineering lab.

Rules:
- Keep experiments small and isolated.
- Do not add unnecessary dependencies.
- Prefer readable code over clever abstractions.
- Add or update tests when changing behaviour.
- Never log API keys, secrets, tokens, or sensitive personal data.
- Each experiment should have a tiny README explaining goal, run steps, and learning.
- Do not introduce large frameworks unless clearly needed.
- Do not use company data, company code, or confidential information.
